# Phaser Template

A Phaser template project with already configured webpack, typescript, tslint and prettier.

# Features

-   webpack
-   webpack-dev-server
-   tslint
-   prettier
-   husky
-   typescript
-   phaser typings

# Installation

Fork this repo.

```
npm install
```
